Description : Develop an Android application that generates QR codes from text or

URLs and allows users to save and share them.

Task Workflow
Step 1: Design a user-friendly interface for text/URL input.
Step 2: Integrate a QR code generation library and
generate QR codes from user input.
Step 3: Display the generated QR code within the
application.
Step 4: Implement functionality to save QR codes as image
files.
Step 5: Add sharing functionality to distribute QR codes
through other applications.

Skills You Will Learn
Android UI Development
Library Integration
File Storage Management
Image Handling
Mobile App Development Fundamentals

Key Features
Text & URL Input
QR Code Generation
Save QR Code as Image
Share QR Code
